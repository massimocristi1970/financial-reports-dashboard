Data Processing Guidelines

Supported File Formats:

  • CSV files (.csv) - Comma-separated values
  • Maximum file size: 50MB per file
  • Maximum rows: 100,000 records per upload
  • Encoding: UTF-8 recommended

Data Format Requirements:

  • Dates: DD/MM/YYYY, YYYY-MM-DD, or MM/DD/YYYY format
  • Currency: Numbers only (e.g., 1000.50) - currency symbols will be stripped
  • Percentages: Numbers only (e.g., 85.5 for 85.5%)
  • Customer IDs: Text or numbers are both supported
  • Headers: First row should contain column headers

Data Storage:

  • IndexedDB: Unlimited storage capacity
  • Append Mode: Add new data to existing records
  • Replace Mode: Replace all existing data
  • Data Persistence: Survives browser restarts
